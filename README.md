# 🍅 番茄钟 (Pomodoro Timer)

一个简洁美观的番茄工作法计时器，帮助您提高专注力，合理安排工作与休息时间。

## 功能特性

- **三种模式**：专注（25分钟）、短休息（5分钟）、长休息（15分钟）
- **进度圆环**：可视化倒计时，剩余时间一目了然
- **循环计数**：每完成 4 个番茄进入长休息
- **自动切换**：专注结束后自动进入休息模式（2秒延迟）
- **声音提示**：倒计时结束时播放提示音
- **桌面通知**：倒计时结束弹出系统通知（需授权）
- **键盘快捷键**：
  - `Space` — 开始 / 暂停
  - `R` — 重置
  - `S` — 跳过当前阶段
- **主题配色**：专注（红色）、短休息（绿色）、长休息（蓝色）自动切换
- **响应式设计**：适配桌面端和移动端

## 技术栈

- 原生 HTML / CSS / JavaScript（无任何第三方依赖）
- CSS 自定义属性实现主题切换
- Web Audio API 生成提示音
- Notification API 发送桌面通知

## 快速开始

直接在浏览器中打开 `index.html` 即可使用：

```bash
open index.html
```

## 使用方法

1. 点击 **「开始」** 按钮或按 `Space` 键开始专注计时
2. 计时结束自动进入休息，或点击 **「跳过」** 手动切换
3. 点击 **「重置」** 重置当前阶段计时
